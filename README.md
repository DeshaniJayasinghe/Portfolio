## ✨ Features
- **Modern UI/UX**: Built with React and Tailwind CSS for a sleek, professional look.
- **Dark Mode Support**: Fully functional Dark/Light mode toggle for better accessibility.
- **Fully Responsive**: Optimized for Mobile, Tablet, and Desktop views.
- **Dynamic Projects Section**: Interactive card layouts to showcase academic and personal projects.
- **Smooth Animations**: Powered by Framer Motion for a premium feel.

## 🛠️ Tech Stack
- **Frontend**: React.js (Vite)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React / Simple Icons
- **Deployment**: GitHub Pages (Custom Domain)

## 🚀 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```
   git clone [https://github.com/DeshaniJayasinghe/Portfolio.git](https://github.com/DeshaniJayasinghe/Portfolio.git)
```

2. **Navigate to the project directory:**

```
cd Portfolio
```

3. **Install dependencies:**

```
npm install
```


4.**Start the development server:**

```
npm run dev
```
